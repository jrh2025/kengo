Workout Timer (體能訓練計時器)
Android Pydroid 3 測試版安裝與使用說明

一、 前言

歡迎您參與「Workout Timer」應用程式的測試。本程式是一款專為間歇性訓練（HIIT）設計的計時器，您可以自訂鍛鍊、休息時間與總次數，並搭配背景音樂，讓您的訓練過程更有效率且充滿動感。

本文件將引導您如何在 Android 手機上，透過 Pydroid 3 環境來安裝並執行此應用程式。請依照下列步驟操作，若遇到任何問題，請隨時回報。

二、 環境準備

在開始之前，請確保您的 Android 手機已連上網路。

步驟 1：安裝 Pydroid 3

開啟您手機上的「Google Play 商店」。在搜尋欄中輸入 Pydroid 3，找到「Pydroid 3 - IDE for Python 3」，點擊並安裝。

步驟 2：在 Pydroid 3 中安裝 Kivy 函式庫

Kivy 是本應用程式的圖形介面框架，必須先安裝才能執行。

  1. 開啟 Pydroid 3 應用程式。
  2. 點擊左上角的選單圖示 (☰)，選擇「Pip」。
  3. 在「LIBRARY NAME」欄位中輸入 kivy。
  4. 點擊「INSTALL」按鈕。
  5. 程式會開始下載並安裝 Kivy。安裝過程可能需要數分鐘，請耐心等候直至出現 "Complete" 或類似的完成訊息。

步驟 3：下載並解壓縮應用程式檔案

所有需要的檔案都已打包成一個壓縮檔，方便您一次下載。

  1. 在您的手機瀏覽器中開啟以下連結，下載應用程式壓縮檔：
    https://drive.google.com/file/d/1es9duAjCFz90lJ7_CVvNmBSMMntC-rMR/view?usp=sharing
  2. 下載完成後，請到您手機的「檔案管理」或「下載」資料夾中找到 WorkoutTimer.zip 這個檔案。
  3. 點擊此壓縮檔，並選擇「解壓縮」或「Extract」。請將其解壓縮至您手機內部儲存空間的根目錄或您熟悉的位置。
  4. 解壓縮後，您會得到一個名為 WorkoutTimer 的資料夾。請確認該資料夾內的結構如下： 
    WorkoutTimer/
├── main.py
├── workouttimer.kv
├── fonts/
│   └── NotoSansTC-Regular.ttf
├── music/
│   └── (此處可能是空的)
└── sounds/
    ├── countdown.wav
    ├── start_work.wav
    ├── start_rest.wav
    ├── finish.wav
    └── background_music.mp3
  5. (重要) 請將您自己喜歡的 .mp3 或 .wav 音樂檔案複製或移動到 WorkoutTimer/music 這個子資料夾中。這些是您稍後可以在 App 裡選擇播放的背景音樂。

三、 執行應用程式

  1. 再次開啟 Pydroid 3。
  2. 點擊畫面中的「資料夾」圖示，開啟檔案總管。
  3. 選擇「Internal Storage」，然後找到並進入您剛剛解壓縮出來的 WorkoutTimer 資料夾。
  4. 點擊 main.py 檔案將其載入。
  5. 點擊畫面右下角的黃色播放按鈕 (▶)，應用程式便會開始執行。

四、 功能使用說明

應用程式啟動後，您會看到主介面。以下是各項功能的詳細說明：

1. 主顯示區

  • 狀態標籤 (State Label)：顯示目前的狀態，例如「準備設定」、「鍛鍊中」、「休息一下」等。
  • 時間顯示 (Timer Display)：以 `分鐘:秒鐘` 的格式顯示剩餘時間。
  • 次數計數 (Rounds Label)：顯示「目前次數 / 總次數」。

2. 時間與次數設定

在計時器處於「準備設定」或「訓練完成」狀態時，您可以設定以下參數：

  • 鍛鍊時間：在分鐘和秒鐘的數字上上下滑動來調整單次鍛鍊的持續時間。
  • 休息時間：同樣透過上下滑動來調整每次休息的持續時間。
  • 總次數：上下滑動來設定總共要執行的鍛鍊回合數。

3. 音樂與音量控制

  • 選擇音樂：點擊此按鈕會彈出一個視窗，列出您放在 music 資料夾中的所有音樂檔案。點擊其中一首即可將其設定為背景音樂。
  • 音量滑桿 (Volume Slider)：左右拖動滑桿可以調整背景音樂的音量。當前音量會以百分比顯示在左側。
  • 靜音按鈕 (Mute Button)：點擊可立即將音樂靜音或取消靜音。

4. 主要控制按鈕

  • 開始 / 暫停 / 繼續 按鈕：
    • 在初始狀態下，點擊「開始」以啟動計時器。
    • 在計時器運行中，此按鈕會變為「暫停」。點擊可暫停計時。
    • 在暫停狀態下，此按鈕會變為「繼續」。點擊可從暫停處繼續。
  • 重設 (Reset) 按鈕：無論在哪個狀態，點擊此按鈕都會立即停止計時器，並將所有設定恢復到初始的「準備設定」畫面。

五、 訓練流程詳解

  1. 設定：首先，設定好您的「鍛鍊時間」、「休息時間」與「總次數」。
  2. 開始：點擊「開始」按鈕。
  3. 準備：畫面會變為藍色，並有 5 秒的準備倒數計時。
  4. 鍛鍊：倒數結束後，畫面轉為紅色，進入「鍛鍊中」狀態，並播放鍛鍊開始的音效。
  5. 休息：鍛鍊時間結束後，畫面轉為綠色，進入「休息一下」狀態，並播放休息開始的音效。
  6. 循環：系統將依照您設定的總次數，重複「鍛鍊」與「休息」的循環。
  7. 完成：所有次數完成後，畫面會恢復初始顏色，顯示「訓練完成！」，並播放完成音效。此時您可以點擊「再次開始」進行新一輪訓練，或點擊「重設」來重新調整參數。

六、 疑難排解

應用程式閃退或顯示錯誤：

  • 最可能的原因是 Kivy 函式庫未成功安裝。請回到「二、 環境準備」的「步驟 2」重新安裝 Kivy。
  • 請確認您已成功解壓縮 WorkoutTimer.zip 檔案，且 WorkoutTimer 資料夾結構完整。

「選擇音樂」按鈕無反應或顯示資料夾是空的：

  • 請確認您已經在 WorkoutTimer/music 資料夾中放入了至少一個 .mp3 或 .wav 檔案。

文字顯示不正常或變成方塊：

  • 請確認 WorkoutTimer/fonts 資料夾中存在 NotoSansTC-Regular.ttf 檔案。
