TaiwanGeoAI 使用說明

個人化台灣 AI 地理探索平台 V1.0

您好！歡迎使用「個人化台灣 AI 地理探索平台」。

這不僅僅是一個地圖軟體，更是一個專為您打造的私人知識庫。您可以把它想像成一本活的、可以無限擴充的台灣百科全書，而您就是這本書的作者。透過強大的 AI 技術，您可以一鍵生成台灣任一鄉鎮的詳細介紹，並自由地修改、補充，加入自己的照片與筆記，打造出一個完全屬於您的數位台灣。

最重要的是，您所有的資料都只會儲存在您自己的電腦上，我們絕不存取，確保您的隱私與安全。

一、首次啟動：設定您的 AI 金鑰

在開始探索之前，我們需要進行一個簡單的一次性設定，讓軟體能與 AI 連接。

  1. 開啟金鑰設定視窗：點擊程式視窗左上角的 設定 菜單，選擇 設定 Gemini API 金鑰。
  2. 輸入您的金鑰：會彈出一個視窗，請將您自己的 Google Gemini API 金鑰貼到輸入框中。

如何取得 Google Gemini API 金鑰？

如果您沒有金鑰，可以依照以下步驟免費取得：

  1. 前往 Google AI Studio：使用您的網頁瀏覽器，開啟官方網站：https://aistudio.google.com/
  2. 登入 Google 帳號：使用您的個人 Google 帳號登入。
  3. 取得 API 金鑰：登入後，點擊頁面上的 Get API key (取得 API 金鑰) 按鈕。
  4. 建立新的 API 金鑰：在新的頁面中，點擊 Create API key in new project。
  5. 複製金鑰：系統會立刻生成一組長字串，點擊旁邊的複製圖示將它完整地複製下來。
  6. 貼回本軟體：回到本軟體的金鑰設定視窗，將剛剛複製的金鑰貼上。
  7. 儲存金鑰：點擊 儲存 按鈕。軟體會將您的金鑰安全地加密保存在您的電腦中。

設定完成後，您就可以開始使用 AI 的強大功能了！

二、基本操作：探索與生成

軟體的主畫面分為兩部分：左邊是互動地圖，右邊是資料顯示區。

選擇鄉鎮

在地圖上，您可以自由縮放與拖曳。當滑鼠移動到某個鄉鎮上方時，該鄉鎮的邊界會高亮顯示。點擊地圖上的任一鄉鎮，右側的資料顯示區就會準備好顯示該鄉鎮的資訊。

AI 一鍵生成資料

選擇一個鄉鎮後，右下角的 AI 一鍵生成資料 按鈕就會變為可用狀態。點擊此按鈕，程式會詢問您是否確定，因為這個操作會覆蓋目前已有的內容。確認後，您會看到「AI 運算中...」的提示。完成後，豐富的圖文資料就會自動呈現在右側面板上。

三、核心功能：編輯您的個人知識庫

AI 生成的資料是一個很棒的起點，但真正的樂趣在於將它變成您自己的東西！

  1. 進入編輯模式：當右側顯示資料時，點擊 編輯 按鈕，顯示的文字就會變成可以修改的輸入框。
  2. 修改內容：
    • 修改文字：直接在文字框中打字，就可以修正 AI 的內容，或加入您自己的見解。
    • 管理列表：對於列表項目，可以點擊 新增...項目 來增加，或點擊項目旁邊的 刪除 按鈕來移除。
  3. 上傳您的照片：在編輯模式下，點擊 新增圖片 按鈕，從您的電腦中選擇照片。照片會以縮圖形式顯示。
  4. 儲存變更：完成所有修改後，請務必點擊 儲存 按鈕。若想放棄修改，可點擊 取消 按鈕。

四、進階功能：管理與分享您的成果

當您累積了豐富的資料後，這些功能會非常實用。

  • 備份與分享 (ZIP)：在頂部 資料 菜單中，選擇 匯出鄉鎮資料成 ZIP...，可將目前鄉鎮的所有文字和圖片打包。相對地，也可透過 匯入鄉鎮資料... 讀取備份檔。
  • 製作講義 (PDF)：在 資料 菜單中，選擇 匯出鄉鎮資料成 PDF...，可將資料製作成一份排版乾淨的 PDF 文件。
  • 調整閱讀舒適度：到 設定 -> 調整字體大小 菜單中隨時調整。
  • (高手專用) 自訂 AI 生成內容：到 設定 -> 編輯 AI 生成欄位... 進行修改。您可以新增、移除、重新排序 AI 生成的項目，甚至修改給 AI 的指令。
