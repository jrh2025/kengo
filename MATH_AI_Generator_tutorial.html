MATH AI 智慧題庫生成器

設計理念與詳細使用手冊

一、設計理念：成為教師的專業 AI 助理

在教育現場，教師們花費大量心力在設計高品質、符合課綱且具鑑別度的評量試题上。這個過程不僅耗時，更需要持續的專業投入與創意發想。「MATH AI 智慧題庫生成器」的誕生，旨在解決這個核心痛點。

我們的目標不是取代教師，而是成為教師最強大的專業 AI 助理。我们相信，AI 的力量在於將教師從重複性的勞動中解放，讓他們能更專注於教學設計、學生互動與因材施教等更具創造性的工作上。

本工具的設計遵循以下核心原則：

  • 結構化指令，確保專業產出：AI 的產出品質，取決於輸入指令的品質。本工具的核心是「設定檔產生器」，它引導使用者輸入結構化、標準化且符合教育專業的參數。
  • 課綱為本，緊扣教學現場：所有試題設計皆以台灣的「108課綱」為基礎，確保每一次的生成任務都能精準對應到特定的學習內容條目。
  • 流程分離，簡化複雜操作：將複雜的 AI 生成任務拆解為兩個獨立且連續的步驟：「1. 設定檔產生器」與「2. AI 試題生成器」，引導使用者完成清晰的工作流程。
  • 透明與彈性，賦予使用者完全控制權：使用者可以隨時下載包含完整指令的「Prompt 文字檔」，並將這份指令拿到任何自己慣用的 AI 平台（如 ChatGPT、Claude 等）上使用。
  • 面向未來的資料結構：下載的 JSON 結果完整保留生成條件，方便未來整合進更大的題庫系統。
  • 實用的輸出格式，無縫接軌工作流程：支援多種匯出格式（DOCX, PDF, HTML），並貼心地自動產生教師版（含答案與詳解）與學生版。

二、詳細使用手冊

程式核心功能簡介

「MATH AI 智慧題庫生成器」主要提供兩大核心功能，您可以根據需求彈性使用：

  • 功能一：專業 Prompt 產生器（無需 API 金鑰）
    位於「1. 設定檔產生器」分頁。此功能專注於協助您依據 108 課綱，產生結構化、高品質的 AI 指令 (Prompt)。
  • 功能二：AI 試題自動生成（需要 API 金鑰）
    位於「2. AI 試題生成器」分頁。此功能會直接連接 Google Gemini AI，執行 Prompt 指令並自動產出試題。

功能一：專業 Prompt 產生器 (無需 API 金鑰)

此功能讓您即便沒有 API 金鑰，也能利用本工具的課綱整合優勢，產出專業的 AI 指令。

  1. 啟動程式與準備：執行 MATH_AI_Generator.exe 檔案來啟動程式。請確保 math_curriculum.json 檔案與執行檔放在同一個資料夾內。
  2. 定義試題規格：在「1. 設定檔產生器」分頁填寫所有欄位。其中「測驗目標」是影響生成品質最重要的欄位！請盡量寫得明確具體。
  3. 產出與使用：點擊「下載 Prompt 文字檔 (.txt)」按鈕，程式會將您的所有設定儲存成一個純文字檔。您可以將這個檔案的內容完整複製，貼到任何 AI 服務的對話框中即可生成試題。生成的試題是包含詳解的json檔，你提供這個json檔繼續讓 AI 為你生成互動式的html檔讓它發揮更多價值。

功能二：AI 試題自動生成 (需要設定 API 金鑰)

如果您想一站式完成所有工作，可以直接使用本程式內建的 AI 生成引擎。

  1. 設定 API 金鑰 (僅需設定一次)：
    • 前往 Google AI Studio 網站，登入您的 Google 帳號，即可免費產生一組 API 金鑰。
    • 在程式的「2. AI 試題生成器」分頁，貼上您的金鑰，勾選「記住金鑰」並點擊「儲存」。
  2. 開始生成：確認設定無誤後，點擊巨大的「⚡ 開始生成試題 ⚡」按鈕。
  3. 查看與下載成果：下方的文字框會顯示詳細的運作日誌與最終的試題預覽。生成成功後，即可使用下方的下載按鈕匯出 DOCX / PDF / HTML 或 JSON 檔案。

三、常見問題 (FAQ)

Q: 我一定要有 API 金鑰才能使用這個程式嗎？

A: 不需要。您可以完全免費地使用「1. 設定檔產生器」的所有功能。只有當您想直接使用「2. AI 試題生成器」的一鍵生成功能時，才需要設定您自己的 API 金鑰。

Q: 當我點擊「開始生成試題」時，程式顯示「生成失敗！」該怎麼辦？

A: 請先查看日誌區塊顯示的錯誤訊息。常見原因有：API 金鑰錯誤、網路問題、或 API 免費額度暫時用盡。

Q: 生成的試題品質不理想，可以怎麼改善？

A: 試題品質的關鍵在於第一分頁的「測驗目標」。請試著將測驗目標寫得更明確、更具體。例如，與其寫「測驗一元一次方程式」，不如寫「測驗學生是否能將應用題的文字情境，正確轉換為一元一次方程式並求解」。
